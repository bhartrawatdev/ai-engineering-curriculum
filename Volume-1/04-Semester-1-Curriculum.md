# Semester 1 — Foundations of Computing & Mathematics

**Total Credits:** 17
**Theme:** Establish programming fluency, developer tooling, and the mathematical language needed for all later AI coursework.

---

## CS101 — Introduction to Programming & Computational Thinking

| Field | Detail |
|---|---|
| Credits | 4 (3 Lecture + 1 Lab) |
| Prerequisites | None |
| Corequisites | CS103 (Programming Lab I) |

**Description:**
Introduces programming fundamentals using Python: variables, control flow, functions, basic I/O, and problem decomposition. Establishes computational thinking as a transferable skill before any AI-specific content is introduced.

**Learning Outcomes:**
- Write, debug, and test Python programs from a written specification.
- Decompose a problem into functions and reusable logic.
- Apply basic control structures (loops, conditionals, recursion) correctly.
- Read and interpret error messages and tracebacks independently.

---

## CS102 — Linux, Git & Developer Tooling

| Field | Detail |
|---|---|
| Credits | 2 (1 Lecture + 1 Lab) |
| Prerequisites | None |
| Corequisites | CS101 |

**Description:**
Hands-on introduction to the Linux command line, filesystem navigation, shell scripting basics, and Git/GitHub workflows (clone, commit, branch, pull request). This course is the practical foundation for every lab and project in the program.

**Learning Outcomes:**
- Navigate and operate confidently in a Linux terminal environment.
- Use Git for version control: commit, branch, merge, resolve basic conflicts.
- Collaborate on a shared repository using pull requests and code review etiquette.
- Write simple shell scripts to automate repetitive tasks.

## CS103 — Programming Lab I

| Field | Detail |
|---|---|
| Credits | 2 (Lab only) |
| Prerequisites | None |
| Corequisites | CS101 |

**Description:**
Applied lab pairing with CS101. Weekly hands-on exercises reinforcing programming fundamentals through small coding challenges, culminating in a mini text-based project.

**Learning Outcomes:**
- Translate lecture concepts from CS101 into working, tested code.
- Build a small end-to-end command-line program independently.
- Practice debugging workflows and basic unit testing.

## MA101 — Calculus for Engineers I

| Field | Detail |
|---|---|
| Credits | 4 |
| Prerequisites | None |
| Corequisites | None |

**Description:**
Limits, derivatives, and integrals of single-variable functions, with engineering applications. Builds the calculus foundation required for later optimization and machine learning mathematics (Semester 3–5).

**Learning Outcomes:**
- Compute derivatives and integrals of standard function classes.
- Apply differentiation to rate-of-change and optimization problems.
- Connect calculus concepts to gradient-based reasoning used later in ML.

---

## MA102 — Discrete Mathematics & Logic

| Field | Detail |
|---|---|
| Credits | 3 |
| Prerequisites | None |
| Corequisites | None |

**Description:**
Propositional and predicate logic, set theory, combinatorics, and proof techniques (induction, contradiction). Provides the reasoning foundation for algorithm correctness and complexity analysis (Semester 3).

**Learning Outcomes:**
- Construct and evaluate formal logical arguments.
- Apply set theory and combinatorics to counting problems.
- Write basic mathematical proofs, including proof by induction.

## Semester 1 Prerequisite Notes

- No course in Semester 1 has a prerequisite — this is the entry point of the program.
- CS101 + CS103 are tightly coupled (lecture + lab) and should be taken together.
- CS102 (Git/Linux) is a hard prerequisite for every lab course from Semester 2 onward.
- MA101 and MA102 both feed directly into MA103/MA104 (Semester 2) and, later, MA201/MA202 (Semester 3).
