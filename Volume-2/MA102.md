## Course Description

MA102 develops the formal reasoning toolkit underlying computer science: logic, sets, combinatorics, and proof. Unlike MA101's continuous mathematics, this course deals in discrete, countable structures — the mathematical language of algorithms, correctness arguments, and (later) probability theory.

## Why This Course Is Needed
Algorithm correctness (CS202), complexity analysis (CS202), and even neural network theory rely on precise logical reasoning and proof technique. Without formal exposure to induction and structured argument, students default to "it seems to work" reasoning, which fails at scale.

## Prerequisite Knowledge Required
Pre-university algebra (assumed entry-level).

## Where It Is Used Later
- **CS201/CS202 (Data Structures II, Algorithm Analysis, Semester 3)** — proof by induction is the standard tool for proving algorithm correctness and analyzing recurrence relations.
- **MA201 (Probability & Statistics, Semester 3)** — set theory and combinatorics are direct prerequisites for probability foundations.
- **AI404 (AI Safety, Alignment & Evaluation, Semester 7)** — formal reasoning underlies rigorous evaluation and specification of AI system behavior.

## Learning Objectives

1. Construct and evaluate propositional and predicate logic statements.
2. Apply set theory operations and reasoning.
3. Solve counting problems using combinatorics.
4. Write valid mathematical proofs, especially by induction and contradiction.
5. Reason formally about relations and functions.

## Weekly Topics / Units

| Week | Topic |
|---|---|
| 1 | Propositional logic: statements, connectives (AND, OR, NOT, IF-THEN) |
| 2 | Truth tables, logical equivalence, tautologies/contradictions |
| 3 | Predicate logic: quantifiers (∀, ∃), translating statements |
| 4 | Proof techniques I: direct proof, proof by contradiction |
| 5 | Proof techniques II: mathematical induction (weak) |
| 6 | Mathematical induction (strong), structural induction (conceptual intro) |
| 7 | Set theory: definitions, operations (union, intersection, complement) |
| 8 | Set identities, power sets, Cartesian products |
| 9 | Relations: properties (reflexive, symmetric, transitive), equivalence relations |
| 10 | Functions: injective, surjective, bijective; composition |
| 11 | Combinatorics I: counting principles, permutations |
| 12 | Combinatorics II: combinations, binomial theorem |
| 13 | Pigeonhole principle and its applications |
| 14 | Introduction to graph theory (vertices, edges, degree) — light preview, deepened in CS201 |
| 15 | Review, buffer, and connections to algorithms/probability |

## Laboratory / Problem-Solving Sessions
Weekly proof-writing workshop (not a coding lab) — peer-reviewed proof exercises.

## Assignments
5 problem sets, each requiring at least one full written proof.

## Mini Projects
- **Proof Portfolio (Weeks 13–15):** Students compile and revise their 3 strongest proofs from the semester based on instructor feedback, demonstrating iterative improvement in rigor — first exposure to the "draft, critique, revise" cycle formalized later in HU401 (Research Methods).
